# Modern Full-Stack React Projects: The Front-End

This project contains the front-end code from the book _Modern Full-Stack React Projects_ by Daniel Bugl, available on
O'Reilly. However, the code has been updated to use TypeScript and the latest versions of various libraries.

## Get started

Create a `.env` file with the following:

```shell
VITE_API_URL_STEM=http://localhost:3000/api/v1/
```

If your back-end is running on a different port, change the `VITE_API_URL_STEM` value accordingly.

Then run the following commands:

```shell
npm install
npm run dev
```
