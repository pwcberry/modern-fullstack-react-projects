# Modern Full-Stack React Projects: The Back-End

This project contains the back-end code from the book _Modern Full-Stack React Projects_ by Daniel Bugl, available on
O'Reilly. However, the code has been updated to use TypeScript and Fastify, as well as updated versions of other
supporting packages.

## Get started

Create a `.env` file with the following:

```shell
DATABASE_URL=mongodb://127.0.0.1:27017/blog
PORT=3000
```
